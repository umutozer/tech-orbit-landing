- [ ] HTML kodlari dogru ve guncel olmalidir
- [ ] HTML etiketleri dogru bir sekilde kullanilmalidir
## CSS Uyumlulugu
- [ ] CSS kodlari dogru ve guncel olmalidir
- [ ] CSS selectorleri dogru bir sekilde kullanilmalidir
## JS Hatalari
- [ ] JS kodlari dogru ve guncel olmalidir
- [ ] JS hatalari dogru bir sekilde yakalanmalidir
## Responsive
- [ ] Sayfa mobil cihazlarda dogru bir sekilde gorunmeli
- [ ] Sayfa tablet cihazlarda dogru bir sekilde gorunmeli
- [ ] Sayfa masaustu cihazlarda dogru bir sekilde gorunmeli
## SEO
- [ ] Meta taglari dogru bir sekilde kullanilmalidir
