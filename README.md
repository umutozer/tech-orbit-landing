# tech-orbit-landing

Premium teknoloji ürünleri için hazırlanan, yüksek etkileşimli ve minimalist tasarıma sahip modern tek sayfa tanıtım sitesi.

## Teknoloji

| | |
|---|---|
| Mimari | react-vite |
| Frontend | React + Vite + Tailwind CSS + Framer Motion |
| Backend | none |
| Veritabani | none |
| Deployment | Vercel / Netlify |

## Kurulum

```bash
cd frontend
npm install
npm run dev
```
